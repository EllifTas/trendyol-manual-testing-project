# Test Case: Valid Search Returns Results

## Test Case ID
TC_SEARCH_001

## Objective
To verify that entering a valid search keyword returns matching product results.

## Preconditions
- User is on the homepage.
- User has access to the search bar.

## Test Data
Keyword: "shoes"

## Test Steps
1. Open the Trendyol homepage.
2. Enter the keyword "shoes" into the search bar.
3. Click the search icon or press Enter.
5. Observe the search results page.

## Expected Result
- The system should display a list of products related to "shoes".
- At least one result should appear.
- Page layout should load without errors.

Not Executed
