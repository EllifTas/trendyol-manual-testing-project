# Test Case: Search with No Matching Results

## Test Case ID
TC_SEARCH_002

## Objective
To verify that the system handles searches with no matching products and displays an appropriate message.

## Preconditions
- User is on the homepage.

## Test Data
Keyword: "abcd1234xyz"

## Test Steps
1. Open the Trendyol homepage.
2. Enter a keyword that is not expected to return any results.
3. Click the search icon or press Enter.
4. Observe the results page.

## Expected Result
- No products should be displayed.
- A message such as "No results found" should appear.
- The page should not show broken layouts.

## Status
Not Executed
