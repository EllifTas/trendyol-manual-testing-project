# Test Case: Add a Single Product to Cart from Product Detail Page

## Test Case ID
TC_CART_001

## Objective
To verify that a user can add a single product to the cart from the product detail page.

## Preconditions
- User is on a valid product detail page.
- The product is in stock.

## Test Data
Product: Any in-stock product

## Test Steps
1. Open the product detail page of an in-stock product.
2. Click the "Add to Cart" button.
3. Open the cart page.
4. Observe the cart contents.

## Expected Result
- The selected product should appear in the cart.
- Quantity should be set to 1 by default.
- Product name, price and thumbnail should be displayed correctly.
