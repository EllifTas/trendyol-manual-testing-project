# Test Case: Remove Item from Cart

## Test Case ID
TC_CART_005

## Objective
To verify that the user can remove an item from the cart.

## Preconditions
- Cart contains at least one product.

## Test Steps
1. Open the cart page.
2. Locate an item in the cart.
3. Click the "Remove" (or trash) icon for that item.
4. Confirm the removal if a confirmation dialog appears.
5. Observe the cart contents.

## Expected Result
- The selected product should be removed from the cart.
- Cart total should be updated accordingly.
- If no items remain, the cart should display an "empty cart" message.

Not Executed
